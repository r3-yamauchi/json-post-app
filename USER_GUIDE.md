# JSON POST Tool - 使用方法

## 概要
JSON POST Toolは、指定したURLにJSONデータをPOSTリクエストで送信し、レスポンスを見やすく表示するWebアプリケーションです。

## 主な機能

### 1. URL設定
- アプリケーション起動時にPOSTリクエスト先のURLを設定
- URLの一部を伏せ字（***）で表示してプライバシーを保護
- 「Change」ボタンでいつでもURL変更可能

### 2. JSON入力エディタ
- **リアルタイムバリデーション**: 入力中にJSON形式をチェック
- **シンタックスハイライト**: 見やすいコードエディタ
- **自動インデント**: Enterキー押下時の自動インデント
- **Tab操作**: Tabキーでインデント、Shift+Tabでアンインデント
- **フォーマット機能**: JSONを整形して見やすく表示
- **ミニファイ機能**: JSONを圧縮して表示
- **テンプレート**: よく使用するJSON構造をワンクリックで挿入
- **サイズ表示**: 入力したJSONのサイズを表示

### 3. レスポンス表示
- **ステータス表示**: HTTPステータスコードと説明
- **表示モード切替**: Formatted（整形）とRaw（生データ）の切り替え
- **レスポンスサイズ**: 受信データのサイズ表示
- **コピー機能**: レスポンスデータをクリップボードにコピー
- **ヘッダー表示**: レスポンスヘッダーの詳細表示
- **リクエスト詳細**: 送信したリクエストの詳細情報
- **エラー詳細**: エラー発生時の詳細な説明

### 4. 高度な機能
- **リクエスト履歴**: 過去50件のリクエスト履歴を自動保存
- **URL記憶**: 設定したURLをローカルストレージに保存
- **タイムアウト処理**: 30秒でリクエストタイムアウト
- **CORS対応**: クロスオリジンリクエストに対応
- **レスポンシブデザイン**: デスクトップ・モバイル両対応

## 使用方法

### 1. 初回起動
1. アプリケーションを開くとURL設定ダイアログが表示されます
2. POSTリクエストを送信したいAPIのURLを入力
3. 「Save」ボタンをクリックして保存

### 2. JSONデータの入力
1. 左側のJSON Inputエリアにデータを入力
2. テンプレートを使用する場合は「Templates...」から選択
3. 「Format」ボタンで整形、「Minify」ボタンで圧縮可能
4. 入力中にリアルタイムでバリデーションが実行されます

### 3. リクエスト送信
1. 「Send POST Request」ボタンをクリック
2. ローディング状態が表示されます
3. 右側のResponseエリアに結果が表示されます

### 4. レスポンスの確認
1. ステータスコードで成功/失敗を確認
2. 「Formatted」「Raw」ボタンで表示形式を切り替え
3. 「📋 Copy」ボタンでレスポンスをコピー
4. 「Response Headers」「Request Details」で詳細情報を確認

## トラブルシューティング

### よくある問題
1. **CORS エラー**: 送信先サーバーがCORSを許可していない場合
2. **ネットワークエラー**: インターネット接続やサーバーの問題
3. **タイムアウト**: 30秒以内にレスポンスが返らない場合

### 解決方法
- サーバー側でCORS設定を確認
- ネットワーク接続を確認
- URLが正しいか確認
- サーバーが稼働しているか確認
